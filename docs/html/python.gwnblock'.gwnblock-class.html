set_timer(self, index, interrupt=True, interval=1, retry=1, nickname1='nick1', nickname2='nick2', add_info=None)

source code 

Sets timer values.

Parameters:
  • index - the timer index position.
  • interrupt - if True interrupts timer event generation until set to False.
  • interval - the time between two successive timer events.
  • retry - the number of events to be generated; once this number is reached, the timer is interrupted until interrupt is set to False.
  • nickname1 - the nickname of the event to be generated after each interval.
  • nickname2 - the nickname of the event to be generated once reached the retry number.
  • add_info - additional info.
