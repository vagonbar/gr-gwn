The GWN basic block constructor.

The GWN basic block implements facilites used by all GWN blocks. A descendent of this class may call this constructor to fix the number of ports, timers and timeouts.

Parameters:
  • blkname - the block's name.
  • blkid - a unique block identifier.
  • number_in - number of input ports.
  • number_out - number of output ports.
  • number_timers - number of internal timers.
  • number_timeouts - number of internal timeouts.
