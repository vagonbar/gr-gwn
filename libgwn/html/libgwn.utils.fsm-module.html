Package fsm

source code

The GWN Finite State Machine (GWN FSM) module.

Module gwnfsm describes the FSM. The example modules show the coding of simple FSMs.

The GWN Finite State Machine implementation is an extension of Noah Spurrier's FSM 20020822, http://www.noah.org/python/FSM/.
