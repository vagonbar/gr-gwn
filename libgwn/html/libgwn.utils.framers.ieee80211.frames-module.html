bin2(bb, bitlen=8, with0b=False)

source code 

Converts number into binary string with 0 padding for specified length.

Unlike builtin function bin(), this function pads with left '0' to reach the specified length in the returned binary string. Default length is 8, excluding initial '0b'.

Parameters:
  • bitlen - length of binary string to return; default 8.
  • with0b - if True, starts binary string with '0b'; default False.
Returns:
a binary string of specified length.

mkbm(beg, end, bm_len, as_int=True)

Makes a bitmask, optionally return as int.

Parameters:

  • end - end position of senquence of 1s plus 1, for correct slicing.
  • bm_len - total length of bitmask.
  • as_int - if True, return an int, otherwise return a binary string. Default True.
Returns:
a bitmask, as a binary string or as int (the default).

addrmac2pkt(addrmac)

source code 

Packs a string MAC address into 6 bytes.

Receives a MAC address in readable, string format such as '6c:62:6d:1c:cf:09' and packs it into 6 bytes. When received data is not a formally valid MAC address, returns '000000'.

Parameters:
  • addrmac - a MAC address in string format such as '6c:62:6d:1c:cf:09'.
Returns:
a MAC address packed in 6 bytes, or '000000' on error.

addrpkt2mac(addrpkt)

source code 

Unpacks 6 bytes MAC address into string.

Receives 6 bytes corresponding to a packed MAC address, unpacks it and presents it as a string formmatted address such as '6c:62:6d:1c:cf:09'. When received data is not a 6B string, returns '00:00:00:00:00:00'.

Parameters:
  • addrpkt - a MAC address packed in 6 bytes.
Returns:
a MAC address in string format such as '6c:62:6d:1c:cf:09', or '00:00:00:00:00:00' on error.
