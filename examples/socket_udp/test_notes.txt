GWN - GNU Radio Wireless Network project

Test notes for UDP socket connection
------------------------------------


=== Transmit and Receive flowgraph on a virtual channel.

1. In a terminal, start sock_sender_test.py
    python sock_sender_test.py

2. In GNU Radio Companion, start ev_tx_rx_socket_udp.grc.

3. In another terminal, start sock_receiver_test.py
    python sock_receiver_test.py

4. In terminal opened in step 1, you should see something like:
  SERVER, received message: SEND message 89
  SERVER, received message: SEND message 90


=== Socket sender

1. In GNU Radio Companion, start sock_sender.grc.

2. In a terminal, start sock_sender_test.py
    python sock_sender_test.py

3. In the terminal, you should see something like:
  SERVER, received message: Socket Sender message in event payload
  SERVER, received message: Socket Sender message in event payload


=== Socket receiver

1. In a terminal, start sock_receiver_test.py
    python sock_receiver_test.py

2. In GNU Radio Companion, start sock_receiver.grc.

3. In the GNU Radio Companion ouput window you should see something like: 

  --- Event Sink id 139991041620688, received ev: DataOut
  Event class name: EventComm
  Nickname: 'DataOut'; Type: 'Data'; SubType: 'DataOut'
  Event dict: {}
    src_addr: 
    dst_addr: 
    payload: SEND message 23
    frmpk: ''
